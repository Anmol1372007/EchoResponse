EchoResponse 🚨

Real-time Emergency Coordination & SOS System ### 🌟 Project Overview

EchoResponse is a tactical emergency management dashboard designed for campuses and residential buildings. It bridges the gap between a person in crisis and emergency responders by providing sub-second SOS synchronization.
🚀 Key Features

    One-Tap SOS: Instant alerts for Fire, Medical, and Security emergencies.

    Real-time Dashboard: Admin view pulses instantly when a new crisis is reported.

    Zero-Refresh Persistence: Powered by Google Firestore for live data updates without page reloads.

    Indoor Precision: Maps incidents to exact Room and Floor numbers.

🛠️ Tech Stack

    Frontend: Next.js 14, Tailwind CSS, Lucide React

    Backend: Google Firebase (Cloud Firestore)

    Deployment: Vercel

🛠️ Local Setup

    Clone the repo:

    git clone https://github.com/Anmol1372007/EchoResponse.git

    Install dependencies:

    npm install

    Add your Firebase keys in a .env.local file.

    Run the project:

    npm run dev
